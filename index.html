⚙️ Configure a chave Ably

Para transmitir imagens entre dispositivos, o Radium Tela precisa de uma chave gratuita do Ably.

  1. Acesse ably.com → crie conta grátis
  2. Crie um App → clique em API Keys
  3. Copie a chave (formato: xxxx.yyyy:zzzz)
  4. Cole abaixo e clique em Salvar
